## 📋 Quick Reference

### Key Features

- ✅ User Authentication & Authorization
- ✅ Contact Form with Email & Database Storage
- ✅ AI-Powered Code Generation
- ✅ Project Management System
- ✅ Admin Interface
- ✅ CSRF Protection & Rate Limiting
- ✅ Responsive Design
- ✅ Comprehensive Testing

### Main Technologies

- **Frontend**: Nuxt.js 3, Vue.js 3, Tailwind CSS
- **Backend**: Node.js, Nitro Server
- **Database**: SQLite with Prisma ORM
- **Authentication**: JWT with HTTP-only cookies
- **AI Integration**: Configurable LLM endpoints
- **Testing**: Cypress, Vitest

### Important Directories

- `/pages` - Application routes and pages
- `/components` - Reusable Vue components
- `/composables` - Vue Composition API functions
- `/server/api` - Backend API endpoints
- `/assets/docs` - This documentation
